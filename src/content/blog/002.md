Machine learning can feel confusing at first because people use a lot of different terms. Three of the biggest ones are supervised learning, unsupervised learning, and reinforcement learning.

They all involve machines learning from data or experience, but the way they learn is different.

## Supervised Learning

Supervised learning is when a model learns from examples that already have the right answers.

For example, if you give a model thousands of emails labeled as spam or not spam, it can learn patterns that will help it classify new emails.

Examples:

- Predicting house prices
- Detecting spam emails
- Classifying images
- Predicting whether a customer will cancel a subscription

## Unsupervised Learning

Unsupervised learning is when the model gets data without labeled answers. Instead of being told what to look for, it tries to find patterns on its own.

For example, a store might use unsupervised learning to group customers based on shopping behavior.

Examples:

- Customer segmentation
- Grouping similar documents
- Finding unusual activity
- Organizing large datasets

## Reinforcement Learning

Reinforcement learning is when a model learns by trying actions and receiving rewards or penalties.

Instead of learning from labeled examples, it learns through trial and error.

A simple example is training an AI to play a game. If it makes a good move, it gets rewarded. If it makes a bad move, it gets punished or receives a lower score.

This is like learning by practice and feedback.

## Conclusion

Supervised learning learns with answers.

Unsupervised learning finds patterns without answers.

Reinforcement learning learns through rewards and mistakes.
